Adding a reminder

Reminders are added via the "Add reminder" dialog, which can be reached in the following ways:

  • Click on the launcher icon "Add reminder"
  • Long-click on the launcher icon "SimpleReminder" and select "Add reminder"
  • Use the Quick Settings tile (has to be first added by clicking the "edit" symbol in the Quick Settings menu)
  • Click the "+" button in the reminders list

Select a time using the clock widget and enter a message. Initially, the selected time refers to the next occurrence of the hour/minute within the next 24 hours. If this is on the next day, this is indicated with a blue "+1" next to the displayed date. To select a date more than 24 hours ahead, use the +/- buttons or click on the date. The "+x" next to the date indicates how many days the selected date is ahead. The date is displayed in orange to indicate that it is chosen manually.

To add the reminder, either click the "Add" button on the keyboard (usually a button on the bottom right, with a symbol or titled "Add") or click "Add" at the bottom of the dialog.

After adding a reminder, the time until it is due is shown in hours and minutes (additional seconds are not displayed). If the reminder is at least 24 hours ahead, the number of (calendar) days and hours is shown (minutes are not shown and rounded towards the closest hour).

Nagging reminders

The switch next to the reminder text makes the reminder "nagging": when due, the notification will be renewed every x minutes, producing the same alert (sound, vibration, ...) as when first shown. This continues until the notification is dismissed.

The repeat interval for the current reminder can be adjusted by long-clicking the switch. The default value can be changed in the settings.

Note that changing the default repeat interval in the settings does not affect already scheduled reminders.

Currently there is no indication in the reminders list or the notification whether a reminder is nagging, this will be added later.

Editing a reminder

To edit a reminder, either click on the notification or the entry in the reminders list.

Reminders list

The reminders list can be reached via the launcher icon "SimpleReminder" or via long-clicking on the "Add reminder" launcher icon and selecting "Show reminders".

The reminders list shows all current and past reminders and allows to edit or reschedule them. In addition reminders can be marked as done or deleted. Long-click on a reminder to select it (and potentially further reminders) and see the available actions (a long click on an action symbol shows its name).

The list consists of three sections:

  • Due reminders (yellow): Reminders due according to the set time; a notification has been shown and the reminder has not been marked as done yet (e.g. by dismissing the notification)
  • Scheduled reminders (green): Reminders with due date in the future, a notification will be shown when the due time is reached; this section is further divided into days for the next 7 days
  • Done reminders (grey): Reminders for which the notification has been dismissed or which have been marked done manually

Do Not Disturb mode

By default, reminders should not be shown in Do Not Disturb mode (however, the notification can be seen as soon as Do Not Disturb is disabled). This behaviour can be changed in the system settings under Notifications -> Do Not Disturb -> Alarms and other interruptions. If you enable the category "Reminders", normal and nagging reminders will be shown even in Do Not Disturb mode, with sound and vibration as configured.

Tips

  • To test how a reminder is shown, add a reminder without changing the date or time. The reminder will be shown immediately.
